Eurolens Dashboard

A web-based macroeconomic dashboard for the EU economic area — aggregating ECB interest rates, HICP inflation, GDP growth, and unemployment data across EU member states. Built to make European economic data accessible, visual, and interactive, with an AI assistant for contextual analysis.

Live Demo (Vercel) Architecture →

Architecture

The frontend SPA talks exclusively to the Express proxy. The proxy normalises Eurostat SDMX and World Bank REST responses and routes AI chat to a configurable LLM provider.

Frontend
React 19 · TypeScript · Vite 6
  • TanStack Query v5 for data fetching & caching
  • Recharts for all chart types (line, area, bar)
  • Tailwind CSS v4 with CSS custom properties
  • React Router v6 — hash-free client routing
  • URL search params persist filters across pages
  • AI chat panel with multi-turn history
Backend
Node.js · Express 5 · Render
  • Single server.js — all routes in one file
  • Fetches Eurostat SDMX JSON-stat format
  • Custom SDMX parser (flat index, strides)
  • Multi-provider LLM abstraction (OpenAI / Anthropic / Gemini)
  • In-memory rate limiting on POST /api/chat
  • pg included as dependency (PostgreSQL ready)

Pages & Routes

Six route-level views. Filters are persisted in URL search params so links are shareable.

PageRouteHook(s)Description
Overview / all hooks KPI cards (MRO rate, deposit rate, EZ inflation, GDP, unemployment) + combined MRO vs inflation chart + country inflation bar chart
InterestRates /rates useECBRates MRO & deposit facility rate line chart with time range selector (1Y / 2Y / 5Y / All) + historical rate data table
Inflation /inflation useInflation Multi-country HICP line chart (up to 5 countries) + latest inflation horizontal bar chart for all 10 EU countries. Filters: country multi-select + time range
GDP /gdp useGDP GDP growth bar chart per country + sparklines showing 10-year trend. Country selector filter
Unemployment /unemployment useUnemployment Multi-country unemployment line chart (seasonally adjusted) + latest rates horizontal bar chart. Filters: country multi-select + time range
Compare /compare useCountryComparison Side-by-side 2-country view: inflation line chart, GDP bar chart (last 5 years), summary stats table with latest values + 5Y averages. URL params: ?a=DE&b=FR

Data Sources

All requests are proxied through the Express backend. No external API calls are made from the browser.

Eurostat SDMX REST

Official EU statistics office. SDMX JSON-stat format parsed server-side with a custom flat-index stride algorithm.

irtstm — rates prc_hicp_manr — HICP une_rt_m — unemployment
World Bank REST JSON

Open development indicators API. Used for annual GDP growth rates for all 10 covered EU countries.

NY.GDP.MKTP.KD.ZG mrv=15 · per_page=500
AI Chat Multi-provider

Pluggable LLM backend. Supports OpenAI, Anthropic, and Google Gemini via env config. Includes system guardrails limiting scope to EU economics.

Rate limited (10 req/min) 20-message history Dashboard context injection

Deployment

Frontend and backend are deployed separately. The backend on Render may cold-start after inactivity.

Frontend
Vercel
Static SPA deployment. Set VITE_API_BASE_URL to your Render backend URL in Vercel environment variables.
Backend
Render
Node.js web service. Requires LLM_PROVIDER, LLM_API_KEY, and optionally LLM_MODEL set as environment variables. Free tier cold-starts apply.
Environment variables
backend/.env
PORT=3000
LLM_PROVIDER=# openai | anthropic | google
LLM_API_KEY=
LLM_MODEL=# optional
frontend/.env
VITE_API_BASE_URL=https://your-backend.onrender.com
